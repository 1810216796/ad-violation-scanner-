# ad-violation-scanner-
一套全自动的网站广告违禁词检测工具，包含 Python 爬虫扫描、Shell 进程守护、PHP 实时监控面板。

## 功能特性

- **违禁词库**：覆盖广告法绝对化用语、医疗功效、金融承诺等 200+ 关键词，并支持上下文白名单，避免误判。
- **递归爬取**：多线程并发扫描，自动提取同域名下所有页面，过滤静态资源。
- **增量/全量**：默认增量扫描（断点续扫），每日自动触发全量重置，确保无遗漏。
- **进程守护**：每分钟监控扫描进程，若卡死或退出则自动重启。
- **实时监控**：PHP 面板每秒刷新，展示今日扫描页数、违规数、站点状态、违禁词聚合。
- **自动报告**：扫描结束后生成 HTML 汇总报告，方便存档。
